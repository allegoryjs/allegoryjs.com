AllegoryJS

Simulation-First Interactive Fiction Engine

AllegoryJS bridges the gap between parser-based narratives and systemic game design. It treats the world as a database, not a narrative tree.

Core Architecture

ECS Data Model

World state is managed by a physics/logic simulation using Entity-Component-System architecture.

Semantic Input

Intent is separated from execution. User input is resolved via Vector Embeddings, not rigid string parsing.

Event-Driven

The engine functions as a Request Processor, idling until a user Input or System Event occurs.

Web-Native

Built on standard TypeScript/ESM. No proprietary compilers required.
